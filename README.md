## ✨ Sobre o Projeto

Este portfólio foi criado para funcionar como uma vitrine digital, facilitando o contato com recrutadores e profissionais da área de tecnologia.  

Nele, você encontrará:

- 🎯 **Banner** – Seção de boas-vindas.
- 👨‍💼 **Sobre Mim** – Um resumo da minha trajetória, formação e interesses.
- 🧠 **Projetos** – Galeria com meus principais projetos, contendo descrições, tecnologias e links para GitHub/deploy.
- 🎓 **Cursos** – Certificados e formações complementares.
- 📬 **Contatos** – Meus links profissionais e formas de contato direto.
